1 Project Architecture

  • Full-stack Setup: Explains that the project separates the frontend (client interface) from the backend (server API) to mimic production-grade software.
  • FastAPI Backend: Chosen for its speed, simplicity, and built-in interactive OpenAPI (Swagger) documentation, making backend testing extremely easy.
  • Modern Gemini SDK: Uses the latest google-genai package with the gemini-2.5-flash model to guarantee fast response times and precise analyses.

2 Document Parsing & AI Analysis

  • PDF Text Extraction: The python script uses pypdf to read the uploaded resume bytes in-memory and compile clean string data.
  • Structured Output: Rather than returning plain text, we enforce a strict JSON schema using a Pydantic model.
  • Reliability: Defining a response_schema guarantees the model always yields scores, strengths, weaknesses, and skill gaps in the exact shape the frontend expects.

3 Premium Frontend Design

  • Glassmorphic UI: Utilizes native CSS backdrop-filter and modern transparency properties for a premium, clean design.
  • Performance & Animations: Form validations and file drag-and-drop triggers provide instant, responsive user feedback.
  • Interactive Gauges: Dynamic SVG animations load the matching score, making the analytics feel engaging and alive during live tests.
